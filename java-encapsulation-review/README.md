# Java カプセル化 復習問題

Java初学者向けのカプセル化復習教材です。Eclipseに `Main.java` と各クラスをコピペして実装・確認できる構成です。

## ファイル

- `Javaカプセル化_復習問題.docx`: 生徒配布用。問題文、未完成クラス、動作確認用 `Main.java` を掲載。
- `Javaカプセル化_復習問題_解答.docx`: 講師用。解答コードと実行結果の目安を掲載。
- `lesson-plan-encapsulation.md`: 問題仕様のMarkdown版。
- `assets/bank-kun-implementation.png`: 問2で使う銀行くんイラスト。

## 内容

- 問1: `Student` クラスのフィールドを `private` にし、getter/setterを作成する。
- 問2: `BankAccount` クラスを作成し、残高操作とパスワードを含むカプセル化を学ぶ。
- 問3: 発展問題として、HP・防御力・敗北状態を持つ `Player` クラスを作成する。
